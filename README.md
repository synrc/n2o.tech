Welcome to N2O.TECH!
====================

<img src="https://n2o.dev/DEV.png?v=1"/>

N2O TECH defines the full-stack specification along with reference
implementations in functional programming languages. It provides
WebSocket connectivity with minimalistic enterprise protocol stack:
web framework, business process management, scalable data storage.

Client Languages
----------------

* Client Tier 3：（JavaScript、Lua）
* Client Tier 2：（Swift、Kotlin、TypeScript）
* Client Tier 1：（UrWeb, OCaml）
* Client Tier 0：（Formality）

Server Languages
----------------

* Server Tier 3：（PHP、Python、Perl、Ruby）
* Server Tier 2：（Erlang、Elixir）
* Server Tier 1：（Standard ML、Haskell、F#、Rust）
* Server Tier 0：（Coq、Agda、Lean）
